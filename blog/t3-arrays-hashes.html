Arrays and Hashes. Why?

June 1, 2015

The Array

Let's begin with arrays. An array is an ordered collection of objects. These objects can be a multitude of things. They can be Strings, Integers, Floats, Arrays, etc. The objects in an array are based on a consistent, consecutive numerical index. The first object in the array is located at the '0' position of the array. The next object is at index '1'. Then, index '2' and so on. The picture below demonstrates how objects are organized in an array.
It is important to remember that the contents of an array always remain in the same order unless you move them around. You can do this in a few ways. But, it involves adding or removing an object.
You can create an array in one of four ways:
  • Using Array.new
  • The literal way: array = []
  • With a top-level method called Array
  • With the special %w{...} and %i{...} notations
["this", "that", "1", 1, 2.0, [4,3]]
The picture above is what an array looks like in Ruby. As you can see, the array has strings, an integer, a float, and an array. "this" is located at index '0' of the array. You can use this property to your advantage if you were looking for every other object in the array. You can achieve this by iterating through the array and selecting the objects located in the 'even' indices. Or say you wanted to remove the first object in the array. You already know the first object is at index '0'. So you can simply tell the computer to remove the object at that location.

The Hash

Like an array, a hash is also a collection of objects. However, instead of organizing the objects in a linear way, the hash consists of key/value pairs. You can access the values through their corresponding keys.
The picture above shows a hash with one key/value pair. You can simply obtain the value, "Alex Tamoykin", with the "name" key. Hashes in Ruby do remember their key-insertion order. This means that you can iterate through it similarly to an array. But this is seldomly used since arrays are perfect for that.
Like an array, you can create a hash in one of four ways:
  • Using Hash.new
  • The literal way: hash = {}
  • With a top-level method called Hash
  • With the Hash.[] method
The picture above shows how to create and access a hash in Ruby. To access the value, you would use the variable followed by the key within the brackets.
A good use for a hash would be if you needed to keep a library of the ages of a particular group of people. In this case the key would be a person's name. And the value would be that person's age. When the time comes and you need to get a specific person's age, you can just call up the key with the name and retrieve the age.
