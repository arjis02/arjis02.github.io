The Array

Let's begin with arrays. An array is an ordered collection of objects. These objects can be a multitude of things. They can be Strings, Integers, Floats, Arrays, etc. The objects in an array are based on a consistent, consecutive numerical index. The first object in the array is located at the '0' position of the array. The next object is at index '1'. Then, index '2' and so on.
